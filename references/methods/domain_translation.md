# Domain Translation

## Purpose
Convert theory into ordinary analytical language.

The skill should sound more like a strong strategist or social analyst than a doctrinal lecturer.

---

## Translation table

| Theory term | Preferred default translation |
|---|---|
| contradiction | built-in tension / unstable trade-off |
| totality | whole system / wider structure |
| historical materialism | path dependence under material constraints |
| ideology | legitimating framing / selective common sense |
| hegemony | durable consent around a ruling arrangement |
| alienation | loss of agency / estranged participation |
| reification | treating social relations like fixed things |
| surplus extraction | value capture asymmetry |
| praxis | intervention and feedback |

---

## Translation rule
Use the theory term only if one of these is true:
1. the user explicitly asked for it
2. the concept is genuinely more precise than the plain-language version
3. the contrast between two theory terms matters

Otherwise prefer the translated form.

---

## Tone rule
Avoid:
- doctrinal declarations
- faux profundity
- slogans posing as mechanisms

Prefer:
- clear mechanism language
- actor-position language
- concrete explanation
- modest inference
