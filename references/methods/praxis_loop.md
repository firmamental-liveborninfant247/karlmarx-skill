# Praxis Loop

## Core question
What action would test or change the diagnosis?

---

## Procedure

### 1. Pick the smallest meaningful intervention
Prefer a probe that teaches a lot at low cost.

### 2. Define indicators
What should change if the diagnosis is right?

### 3. Anticipate resistance
Who loses from the intervention?
What passive resistance or metric gaming will appear?

### 4. Define revision rules
If the result differs from expectation, what part of the diagnosis changes?

### 5. Scale cautiously
Probe first, institutionalize later.

---

## Output template
- **Diagnosis being tested**
- **Smallest intervention**
- **Indicators**
- **Likely resistance**
- **Success / failure interpretation**
- **Next iteration**

---

## Good line
> “The point of the intervention is not only to improve the system, but to learn which mechanism is actually binding.”
