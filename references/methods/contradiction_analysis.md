# Contradiction Analysis

## Core question
What mutually dependent pressures pull the system in opposing directions?

---

## Procedure

### 1. Name the pressures precisely
Bad:
- freedom vs control

Better:
- low-friction scaling vs trust-preserving moderation
- productivity quotas vs care quality

### 2. Show dependence
Explain why each side needs the other.
A contradiction is stronger when the system requires both.

### 3. Identify the principal contradiction
Which tension currently organizes the others?

### 4. Distinguish management from transformation
- management = stabilizes the tension temporarily
- transformation = changes the relation that reproduces it

### 5. Look for threshold conditions
When does the contradiction intensify, diffuse, or invert?

---

## Output template
- **Pressure A**
- **Pressure B**
- **Why they co-exist**
- **What reproduces both**
- **Which contradiction is principal**
- **What change would transform rather than merely manage it**

---

## Good warning line
> “This tension is not just an unfortunate trade-off; the current arrangement actively reproduces both sides.”
