# 专门的 AI Skill 发布平台

## 主流平台

### 1. OpenAI GPT Store ⭐ 最推荐
- **网址**: https://chat.openai.com/gpts
- **类型**: 官方 Skill 商店
- **用户量**: 3亿+ ChatGPT 用户
- **你的操作**: 
  1. 打开 ChatGPT → Explore GPTs → Create
  2. 复制 SKILL.md 到 Instructions
  3. 上传 references/ 文件到 Knowledge
  4. 设置名称、描述、Logo
  5. 发布为 Public
- **优势**: 最大流量，用户直接可用，有分类和搜索
- **缺点**: 需要 ChatGPT Plus ($20/月)

### 2. Poe (by Quora)
- **网址**: https://poe.com
- **类型**: 多模型 Bot 平台
- **你的操作**:
  1. 创建 Bot
  2. 粘贴 SKILL.md 作为 Base Prompt
  3. 设置为 Public
  4. 选择模型 (Claude-3.5-Sonnet 推荐)
- **优势**: 
  - 免费创建和分享
  - 支持 Claude、GPT、Gemini 多个模型
  - 有发现页和排行榜
  - 用户可以 Fork 你的 Bot
- **缺点**: 知名度不如 GPT Store

### 3. FlowGPT
- **网址**: https://flowgpt.com/prompt
- **类型**: 提示词/Skill 市场
- **你的操作**:
  1. 创建 Prompt
  2. 粘贴 SKILL.md
  3. 设置价格和封面
  4. 发布到市场
- **优势**: 
  - 可以设置付费 ($1-$50)
  - 有打赏功能
  - 社区活跃
- **缺点**: 主要是提示词，不是完整的 Skill 系统

### 4. PromptBase
- **网址**: https://promptbase.com
- **类型**: 付费提示词市场
- **优势**: 可以卖钱 ($2.99-$9.99)
- **缺点**: 一次性购买，不是持续的 Skill 服务

## 新兴平台

### 5. Agent.ai
- **网址**: https://agent.ai
- **类型**: AI Agent 市场
- **特点**: 可以创建更复杂的 Agent 工作流

### 6. You.com
- **网址**: https://you.com
- **类型**: AI 搜索引擎 + Agent
- **你的操作**: 创建 YouAgent 并分享

### 7. Coze (字节跳动)
- **网址**: https://www.coze.com
- **类型**: Bot 开发平台
- **特点**: 
  - 类似 GPTs
  - 可以发布到 Discord、Telegram、Slack
  - 免费使用 GPT-4

### 8. Dify
- **网址**: https://dify.ai
- **类型**: LLM 应用开发平台
- **特点**: 开源，可以自托管，也可以分享到他们的市场
