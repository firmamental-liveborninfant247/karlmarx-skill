# Quickstart for Local Skills / Prompt Systems

## Minimal deployment
Use these files first:
- `skills/marxist-methodology/SKILL.md`
- `skills/marxist-methodology/INDEX.md`
- `references/methods/general_problem_router.md`
- `references/methods/foundational_moves_selector.md`
- `references/methods/runtime_answer_contract.md`

## Stronger deployment
Also load:
- foundational files most relevant to your use cases
- benchmark tasks
- validation protocol

## Recommended practice
Do not ask the model to perform all methods at once.
Route the problem first, then select only the foundational and method files that materially sharpen the answer.
