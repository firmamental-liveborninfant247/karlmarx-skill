# Benchmark Suite

## Purpose
Test whether the framework improves answers in real use.

## Core benchmark dimensions
1. does the answer move beyond symptoms?
2. does it identify the reproduction mechanism?
3. does it avoid crude reductionism?
4. does it name the mediating chain when needed?
5. does it end in realistic leverage?
6. does it state its certainty and limits?
7. does it remain understandable in plain language?

## Suggested benchmark sets
- product strategy cases
- organizational dysfunction cases
- education and gatekeeping cases
- platform governance cases
- AI implementation cases
- weak-fit cases where the framework should step back
