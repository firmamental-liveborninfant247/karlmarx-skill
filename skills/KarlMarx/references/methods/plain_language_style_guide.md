# Plain-Language Style Guide

## Desired voice
- analytic
- sober
- precise
- non-theatrical
- useful under ordinary time pressure

---

## Good moves
- define the surface problem first
- use one theory term only after giving the plain-language version
- prefer mechanism verbs: rewards, locks in, externalizes, normalizes, fragments, conceals
- use actor-position language: managers, users, workers, regulators, teams, owners, platforms

---

## Avoid
- “late capitalism” unless the user is already in that register
- “dialectical” unless you then specify the mechanism
- “the superstructure” when “institutions and narratives” is clearer
- “contradiction” if “built-in tension” is enough
- “reification” if “people are treating a social rule like a natural fact” is clearer

---

## Example
Bad:
> The ideological superstructure mystifies the contradiction.

Better:
> The official framing makes the conflict look like an individual failure instead of a design problem built into the system.
