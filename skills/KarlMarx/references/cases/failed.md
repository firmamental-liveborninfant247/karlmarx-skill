# Failure Modes and Historical Misuses

## Why this file exists
A top-tier skill must remember not only what a tradition explains well, but also how it has gone wrong.

---

## Recurring failure modes

### 1. Mechanical determinism
Explaining history as if one track is guaranteed.

### 2. Crude reductionism
Flattening law, culture, psychology, and politics into direct reflections.

### 3. Vocabulary without mechanism
Using “contradiction,” “dialectics,” or “ideology” as ornaments.

### 4. Political-moral substitution
Replacing analysis with denunciation.

### 5. Conspiratorial ideology critique
Treating social misrecognition as necessarily centrally planned deception.

### 6. Bad transfer
Moving a concept from nineteenth-century political economy to a modern domain without adaptation.

---

## Runtime lesson
The package should carry an internal memory that:
- sophistication is not the same as severity,
- theory is not an excuse to stop checking mechanisms,
- and the right amount of Marxist framing is often **less** than the maximum possible amount.
