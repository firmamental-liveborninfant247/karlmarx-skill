# Historically Successful Uses

This file is not a triumphalist list. It records where Marxist-derived analysis has often been illuminating.

## Strong domains
- political economy of capitalism
- labor process analysis
- class and dependency structures
- ideology and legitimation
- historical analysis of institutional transformation
- critique of commodification and value capture

## Why this matters for the package
These successes justify using Marxist-derived tools where:
- social structure is central,
- incentives are historically organized,
- and surface appearances are misleading.

The package should borrow confidence from these domains only where there is real structural similarity.
