# Source-Note Convention

## Purpose
Prevent the package from presenting disputed claims as settled truths.

This is a lightweight scholarly discipline layer, not a full bibliography system.

---

## Four source-note types

### 1. Canonical text support
Use when the claim is clearly grounded in classic Marxist texts or major historical figures in the tradition.

Tag mentally as:
- **[Canonical]**

### 2. Scholarly controversy
Use when major interpretations diverge.

Tag mentally as:
- **[Contested Reading]**

### 3. Modern reinterpretation
Use when later Marxist or post-Marxist thinkers significantly reshape the concept.

Tag mentally as:
- **[Later Development]**

### 4. Domain extrapolation
Use when a concept is being applied outside its original setting.

Tag mentally as:
- **[Applied Extension]**

---

## Practical rule
Before using a concept, ask:
1. Is this clearly canonical?
2. Is there a major dispute I should acknowledge?
3. Am I using a later tradition rather than Marx himself?
4. Am I extending the concept into a modern domain?

If 2, 3, or 4 is yes, certainty usually needs to be reduced.

---

## Examples

### Historical materialism
- [Canonical] Material conditions matter for social explanation.
- [Contested Reading] How strongly they determine non-economic life is disputed.
- [Applied Extension] Using it as “path dependence under material constraints” is legitimate but simplified.

### Ideology
- [Canonical] Ideology can stabilize domination.
- [Contested Reading] The term has critical and descriptive uses across Marxist traditions.
- [Later Development] Gramsci and Althusser are not saying exactly the same thing.

### Alienation
- [Canonical] Marx clearly uses the concept in early writings.
- [Contested Reading] Whether it remains central in the same way in mature Marx is debated.
- [Applied Extension] Using it for modern platform work is plausible but interpretive.

---

## Bottom line
The package should sound confident only where the scholarship is actually firm.
